# Política de Privacidad de MV Premium

**Última actualización:** 18 de febrero de 2026

Esta política describe cómo la extensión **MV Premium** (ID: `aekifnbjmpmbknechejmeanadooapfek`) recoge, usa, almacena y comparte datos.

## 1. Responsable

MV Premium es una extensión de navegador para Mediavida.com mantenida por su desarrollador.

## 2. Qué datos se tratan

MV Premium trata datos en dos grupos:

1. **Datos almacenados localmente en tu navegador** (`browser.storage.local` y, para cachés de rendimiento visual, `localStorage`):
- Configuración de la extensión (tema, toggles, atajos, etc.).
- Contenido creado por el usuario dentro de la extensión (por ejemplo, borradores, plantillas, hilos guardados, posts anclados, palabras silenciadas y personalizaciones).
- Datos de uso local opcionales para estadísticas/heatmap (por ejemplo, acciones de crear/editar/publicar, título/subforo/URL cuando aplica).
- Claves API configuradas por el usuario para integraciones opcionales (por ejemplo: Gemini, Groq, ImgBB).

2. **Datos enviados a servicios de terceros solo cuando usas funciones concretas**:
- **Subida de imágenes**: si usas esa función, se envía la imagen (base64/archivo) a ImgBB o freeimage.host.
- **Funciones de IA**: si usas resúmenes u otras funciones IA, se envía el prompt/contenido que tú dispares a Gemini (Google) o Groq.
- **Búsqueda/consulta de contenido multimedia**: consultas de GIFs, cine/series, juegos o tweets ligeros pueden enviar términos, URLs o parámetros técnicos a GIPHY, TMDB, Steam, IGDB/Twitch y endpoints públicos de X/Twitter.

## 3. Finalidad del tratamiento

Los datos se usan para:
- Proporcionar funciones de la extensión en Mediavida.
- Guardar tus preferencias y contenido local.
- Ejecutar integraciones externas bajo demanda del usuario.
- Mejorar la experiencia local (por ejemplo, cachés visuales y estado entre páginas).

## 4. Cómo se recogen los datos

Los datos se recogen:
- Cuando configuras opciones en la extensión.
- Cuando usas acciones específicas (ej. subir imagen, resumir con IA, buscar GIF).
- Cuando navegas por Mediavida y una función necesita leer contexto de la página para funcionar.

## 5. Cómo se comparten los datos

MV Premium **no vende datos**.

Los datos solo se comparten con terceros cuando es necesario para la función solicitada por el usuario, por ejemplo:
- ImgBB / freeimage.host (subida de imágenes)
- Google Gemini / Groq (generación IA)
- GIPHY, TMDB, Steam, IGDB/Twitch, X/Twitter (consultas de contenido)

Cada proveedor externo aplica sus propias condiciones y política de privacidad sobre los datos que recibe.

## 6. Retención y eliminación

- Los datos locales permanecen en tu navegador hasta que:
  - los borres desde las opciones de MV Premium,
  - limpies almacenamiento del navegador,
  - o desinstales la extensión.
- Los datos enviados a terceros quedan sujetos a la retención y políticas de esos terceros.

## 7. Controles del usuario

Puedes:
- Activar/desactivar funciones desde ajustes.
- Activar/desactivar el tracking de actividad local.
- Exportar/importar datos.
- Borrar historial de actividad.
- Eliminar todos tus datos locales desde la zona avanzada de ajustes.

## 8. Seguridad

- MV Premium usa almacenamiento local del navegador para persistencia.
- Las solicitudes de red se gestionan en el contexto de background de la extensión.
- No se incluye venta de datos ni brokers de datos.

Ningún método de almacenamiento/transmisión es 100% infalible, pero se aplican medidas razonables para minimizar exposición de datos.

## 9. Cambios en esta política

Podemos actualizar esta política para reflejar cambios funcionales, legales o de seguridad. La fecha de "Última actualización" se modificará cuando corresponda.

## 10. Contacto

Para dudas sobre privacidad o ejercicio de derechos, contacta con el desarrollador mediante el repositorio oficial:

https://github.com/adangarciadev/mv-premium

# MV Premium Privacy Policy

**Last updated:** February 18, 2026

This policy explains how the **MV Premium** extension (ID: `aekifnbjmpmbknechejmeanadooapfek`) collects, uses, stores, and shares data.

## 1. Controller

MV Premium is a browser extension for Mediavida.com maintained by its developer.

## 2. What data is processed

MV Premium processes data in two groups:

1. **Data stored locally in your browser** (`browser.storage.local` and, for visual performance caches, `localStorage`):
- Extension settings (theme, toggles, shortcuts, etc.).
- User-created content inside the extension (for example: drafts, templates, saved threads, pinned posts, muted words, and customizations).
- Optional local usage data for stats/heatmap (for example: create/edit/publish actions, title/subforum/URL when applicable).
- API keys configured by the user for optional integrations (for example: Gemini, Groq, ImgBB).

2. **Data sent to third-party services only when you use specific features**:
- **Image upload**: if you use this feature, the image (base64/file) is sent to ImgBB or freeimage.host.
- **AI features**: if you use summaries or other AI features, prompted content is sent to Gemini (Google) or Groq.
- **Multimedia/content queries**: GIF, movies/TV, games, or lightweight tweet features may send queries, URLs, or technical parameters to GIPHY, TMDB, Steam, IGDB/Twitch, and public X/Twitter endpoints.

## 3. Purpose of processing

Data is used to:
- Provide extension functionality on Mediavida.
- Save your local preferences and content.
- Run external integrations on user request.
- Improve local UX (for example visual caches and cross-page state).

## 4. How data is collected

Data is collected:
- When you configure extension settings.
- When you use specific actions (for example image upload, AI summary, GIF search).
- When you browse Mediavida and a feature needs page context to work.

## 5. How data is shared

MV Premium **does not sell data**.

Data is only shared with third parties when needed for a user-requested feature, for example:
- ImgBB / freeimage.host (image upload)
- Google Gemini / Groq (AI generation)
- GIPHY, TMDB, Steam, IGDB/Twitch, X/Twitter (content queries)

Each third-party provider applies its own terms and privacy policy to the data it receives.

## 6. Retention and deletion

- Local data remains in your browser until you:
  - delete it from MV Premium settings,
  - clear browser storage,
  - or uninstall the extension.
- Data sent to third parties is subject to those providers' retention and privacy policies.

## 7. User controls

You can:
- Enable/disable features in settings.
- Enable/disable local activity tracking.
- Export/import data.
- Clear activity history.
- Delete all local extension data from advanced settings.

## 8. Security

- MV Premium uses browser local storage for persistence.
- Network requests are handled in the extension background context.
- No data selling and no data broker activity.

No storage/transmission method is 100% infallible, but reasonable measures are applied to minimize data exposure.

## 9. Policy updates

We may update this policy to reflect functional, legal, or security changes. The "Last updated" date will be revised accordingly.

## 10. Contact

For privacy questions or rights requests, contact the developer through the official repository:

https://github.com/adangarciadev/mv-premium
